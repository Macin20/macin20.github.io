从“重庆摄影第一人”开始说起

发表于 2019-07-22 更新于 2021-03-25 分类于 筷子小手
当时签订《烟台条约续增专条》时，清政府耍了小聪明，在条约后面缀了一句：“轮船未抵重庆以前，英国商民不得在彼居住。” 清政府估计是想借长江三峡天险阻碍英国人进入重庆。

上期《除了魔幻8D城市，你还需要了解的重庆》带大家了解了几千年前重庆的样子，那么今天筷子手带你聚焦近百年，触摸重庆近代史的灵魂。

重庆被迫开埠

作为有学问的公众号“筷子小手”的粉丝，想必大家都是集美丽可爱聪慧敏捷于一身的人中“dragon”凤，自然对中学历史课本里1842年中英《南京条约》记忆深刻。清政府鸦片战争战败后，在南京的静海寺签订了中国历史上第一个不平等条约–由此中国沿海沿江共104处，陆续被开放为通商口岸。

而重庆，就包含在这104个通商口岸之中。（重庆是第20个口岸）

重庆摄影第一人

1890年3月，英军在北京和大清朝签订《烟台条约续增专条》，规定“重庆即准作为通商口岸无异”。从条约签订到英国人到达重庆其实还有一段故事，这其中便是手手今日要为大家介绍的这位“重庆摄影第一人”。

她的名字叫做阿绮波德·立德(Archibald Little ) ，她为中国废除裹脚做出了巨大的努力和贡献。我们今天暂不说“裹小脚”的事情，先说说为什么她被称为“重庆摄影第一人”？

根据摄影历史考据，照相机进入中国，是1846年，由供职于中国海关的一位法国人带来。其中最经典，最具代表性的一张老照片，是清朝皇宫里给慈禧拍的像片，就是大家在历史课本中见到的那张。阿绮波德·立德乐在其《穿蓝色长衫的国度》中出版的重庆老照片，比慈禧太后的“标准像”，还要早大约3年。这本书中有10多张照片插页，如像朝天门河滩的竹棚房子，东水门高达5层的文星阁……

这位摄影师不仅爱好写作，而且和她丈夫一样酷爱探险。

当时签订《烟台条约续增专条》时，清政府耍了小聪明，在条约后面缀了一句：“轮船未抵重庆以前，英国商民不得在彼居住。” 清政府估计是想借长江三峡天险阻碍英国人进入重庆。“万万没想到，有人那么快，万万没想到，有人那么厉害。”自以为聪明的清政府万万没有想到这两位酷爱探险的夫妇居然克服了长江天险，自驾船只，就这么登上了重庆的土地。

他们自驾船只开辟长江航道的历史也非常精彩，小手们如果感兴趣可以评论区留言，手手可以专题说说这个故事

1890年7月，重庆海关在朝天门附近正式开关征说，重庆正式开埠。重庆开始与世界亲密接触。

开埠之后

开埠之后，美、英、法、德等国舰船，纷纷驶抵重庆开洋行，办工厂。泊在现今南滨路一带的王家沱、弹子石、龙门浩、海棠溪等沿江码头，装卸并囤积货物。

除立德乐洋行 (这位摄影师夫妇开办的) 外，还有英商的隆茂、太古、白理；法商的聚福、利昌、吉利、安利；德商的瑞记、德昌、宝丰；美商的德士古、美孚；日商的新利、福记、若林等达50余家，其中设行建厂在重庆市南岸区的就有20余家(处)。至今，沿南岸区弹子石至铜元局南滨路一线保留着10多家，所以南岸区是民国时期重庆名副其实的金融商务CBD。有些洋行原址，历经百年风雨，还保存得挺完好的，好像站在跟前就能穿越到那个年代。

零零碎碎了解了一些关于这座城市的过去，絮絮叨叨讲给大家听听。想在说：这座城市的吃喝玩乐之前，尝试着再和它的历史神交一次。

美食虽多，有故事的不多；山河辽阔，看懂了才会透彻。
