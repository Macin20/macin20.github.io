40度！看重庆人如何火炉求生

发表于 2019-08-02 更新于 2021-03-25 分类于 筷子小手
我们对夏天的记忆还算丰富多彩，而我们的下一代是不是对夏天体会最多的就呆在空调房里，拿着冰可乐，感受岁月静好呢？趁记忆还没有完全消失之前，跟着手手，让它在脑海里再停留多一会儿，至少老了以后也有故事对你的孙辈讲。

小伙伴们，你们心中夏天的感觉是什么样的呢？

是正脸对着风扇吹半干的头发，是睡了半程的午觉被窗外的知了吵醒，是切了一半的西瓜拿勺子舀着吃，还是是躺在凉席上妈妈帮你轻轻用蒲扇赶走蚊虫。

我们对夏天的记忆还算丰富多彩，而我们的下一代是不是对夏天体会最多的就呆在空调房里，拿着冰可乐，感受岁月静好呢？趁记忆还没有完全消失之前，跟着手手，让它在脑海里再停留多一会儿，至少老了以后也有故事对你的孙辈讲。

峡谷地形是重庆闷热的元凶。重庆市地貌特征山多河多，地势沿山脉、河流起伏，南北高、中间低，构成以山地、丘陵为主的地表形态，同时形成峡谷，热量汇聚在中间，难以散去。

清泉井水冰镇

暑气蒸足，天光灼背，古代时候，井水就是天然冰窖。所以曹丕才会说，“浮甘瓜于清泉，沉朱李于寒水”，这种清泉井水镇过的西瓜和李子，摸着就是凉丝丝的慰藉，一口下去，既不会有冰箱出来的那种牙齿打颤的感觉，却又让人透心凉。

还有一种诞生于川渝地区的夏日甜品更是重庆人的心头好，那就是冰粉凉虾。据考，冰粉始于明清时期的武阳（今彭山县），在文革时期达到鼎盛。那时川渝地区很多人家里都有几株冰粉树，家家户户夏天都会搓冰粉吃。手手记忆中夏天的冰粉总有一种薄荷味牙膏的味道，不知道大家记忆中的冰粉是什么味道呢？

黄桷树乘凉

记忆中的夏天的蝉总是疯狂歌唱，吟诵生命的短暂与壮烈。既然无论如何都避免不了炎热的天气，那么重庆市的市树黄桷树自然就承担起来为大家遮阴的重任。树叶遮阴，花朵扬馨，黄桷兰的味道也是夏天的味道，黄桷兰喜好温暖的地方，只要温度适宜便可一年四季花开烂漫，夏初到秋初都可以闻到这独特的香气，等到七月流火才渐渐淡去，蝉声静寂就又度过了一个夏天。

防空洞里有乾坤

重庆的洞子有多少，天然的人工的加起来数不胜数。抗战时期防空洞为重庆人民挡住了炮火，和平年代为重庆人民挡住了来自太阳的折磨。只是现在很多防空洞都出于安全原因封闭了，小伙伴有人知道哪些防空洞还在作为避暑纳凉的地方营业吗，快在评论里分享给热成汪的小伙伴们吧。

流水相伴一夏爽

有没有一种娱乐方式，既可以满足大人们喝茶打麻将的乐趣，又可以安抚小朋友们躁动的灵魂。答案当然是“必须有”，万能的手手觉得大家一定会喜欢在分分钟被点燃的夏日炎炎，把双足浸润在流动的溪水中，手上搓着“血战到底”，时不时呷一口菊花茶，然后不露声色地说“胡了”。小朋友们也可以找到自己的小伙伴，玩水枪捉小鱼小虾，单就是嘻嘻闹闹中不知不觉就度过了半晌，躲过了毒辣的阳光。仿佛就是你在笑，他在闹，恨不得须臾间已经过了一生一世。

更刺激的溪降

作为爱刺激爱冒险的筷子手，我们自然要玩的自然要比古人的更刺激更爽。溪降这个名词想必小伙伴们还有些陌生，这种运动在1996年6月才由法国探险家克尼格介绍至中国。手手这次体验的金刀峡溪降，是自上而下由瀑布主体缘绳下降和顺水滑降为主。而手手6月在恩施鹤峰县体验的溯溪则与之相反，是顺沿溪谷逆流而上，克服地形障碍，寻水之源而登山之巅。

总体而言呢都很凉快，都会全身湿透透，溪降对于身体素质的要求更高一些，全程紧绷神经三个小时，毕竟我不会告诉你今天的手手是忍着全身的酸痛在码字的。如果不会游泳的小伙伴需要克服心理障碍，毕竟从近十米的岩石上飞跃而下，除了勇气你还需要泳技。

说了那么多降暑降温法，其实除了救你一命的空调以外，心静自然凉也是必不可少的。

“更宜调息净心，

常如冰雪在心，

炎热亦于吾心少减；

不可以热为热，更生热矣。”
